# 套用[vue-admin-template](https://github.com/woai3c/vue-admin-template.git)基础模板
## ylz-demo-front 考核项目前端部分 [后端部分](https://github.com/lingfenghu/ylz-demo-web)
### 使用
======================
###### 下载
```
cd ylz-demo-front
npm i
```
###### 开发
```
npm run serve
```
###### 打包
```
npm run build
```
======================
### 项目特点
* 导航菜单栏
* 图片验证码
* token登录验证
* swagger接口说明文档
* 侧边栏
* 面包屑
* echart图表展示
* ...
